Overview

simple_xlsx is part of the simple_* ecosystem of focused, single-purpose Eiffel libraries with full Design by Contract support.

Quick Start

Installation

Set environment variable and add to your ECF:

-- Set SIMPLE_EIFFEL=D:\prod
<library name="simple_xlsx" location="$SIMPLE_EIFFEL/simple_xlsx/simple_xlsx.ecf"/>

Basic Usage

local
    l_obj: SIMPLE_XLSX
